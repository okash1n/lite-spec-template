# AGENTS.md - AI Agent Guidelines

このリポジトリで作業する人・AIエージェント向けのガイドラインです。

## 基本ルール

- 仕様は `.litespec/SPEC.md` を最優先する
- 仕様に影響する変更は `.litespec/SPEC.md` を更新する
- 重要な技術判断は `.litespec/adr/` に記録する
- 不明点や矛盾があれば作業前に質問する

## 作業フロー

1. `.litespec/SPEC.md` を読む
2. 仕様に不足があれば追記/提案する
3. 実装する
4. テスト/検証する
5. 変更点を `.litespec/SPEC.md` / `.litespec/adr/` に反映する

## ドキュメント方針

- ドキュメントは日本語で書く
- 仕様・判断・運用ルールの3点を揃える

## テンプレ利用時の注意

- プロジェクトに合わせてこのファイルを調整する
- 運用ルールが固まったら `AGENTS.md` を更新する
