# Lite Spec

Kiro ほど重くない「仕様管理の最低限セット」を使い回すためのテンプレートです。
新しいプロジェクトの直下に配置して、仕様・意思決定・運用ルールを最小コストで揃えることを目的にしています。

## 含まれるもの

- `.litespec/SPEC.md`: 1ページ仕様（目的/範囲/要件/DoD など）
- `.litespec/adr/0001-template.md`: ADR（意思決定ログ）テンプレート
- `AGENTS.md`: 人・AI エージェント共通の運用ルール（ルート）

## 推奨フロー

1. `.litespec/SPEC.md` を埋める
2. `AGENTS.md` をプロジェクトに合わせて調整
3. 重要な判断が出たら `.litespec/adr/` に記録

## 参考

- このテンプレートは「軽量な共通言語」を狙っています
- Kiro の思想は活かしつつ、運用コストを抑える設計です
